PHP TestFest Brasil 2017!

Um grande evento para contribuir com testes do PHP


Setembro - Dezembro 2017

#PHPTestFest #PHPTestFestBrasil2017

Como Você Pode Ajudar

A ideia é juntar em várias cidades do país entre setembro de dezembro de 2017 e, seguindo uma programação pré estabelecida, ensinar como fazer testes para a linguagem PHP. Será disponibilizada uma máquina virtual (VM) para que todos tenham o mesmo ambiente de testes.

Você não possui muita experiência em C ou no próprio PHP? Não tem problema, pois ensinaremos como você pode contribuir, basta ter vontade de aprender.

Quer mais detalhes de onde você pode participar? Acesse os links abaixo.
